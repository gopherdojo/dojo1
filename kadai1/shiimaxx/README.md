# image-convert

## Usage

```
image-convert -s <source extension> -d <destination extention> <directory>
```

```
Usage of image-convert:
  -d string
        destination extension(Short)
  -dest string
        destination extension
  -s string
        source extension(Short)
  -src string
        source extension
  -version
        print version information
```
