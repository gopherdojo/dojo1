goimgconverter
====

Image convert cli.

## Installation

```console
$ go get -u github.com/ohsawa0515/goimgconverter
```

## Usage

```console
$ goimgconverter -h
Usage of goimgconverter:
  -d string
        path of conversion destination
  -f string
        image extension before conversion
  -t string
        image extension after conversion
```
