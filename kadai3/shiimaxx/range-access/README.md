# gurl

gurl is a like curl command by go

## Usage

```
gurl [(-o | --output) <file>] [(-p | --parallel) <parallel>] <url>
```

```
Usage of gurl:
  -o string
        output file(Short) (default "./")
  -output string
        output file (default "./")
  -p int
        number of parallel(Short) (default 10)
  -parallel int
        number of parallel (default 10)
  -version
        print version information
```
