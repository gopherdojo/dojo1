# typing-game

## Usage

```
typing-game [(-n |--num) number of questions] [(-t |--timeout) timeout]
```

```
Usage of typing-game:
  -n int
        number of questions(Short) (default 100)
  -num int
        number of questions (default 100)
  -t int
        timeout(Short) (default 60)
  -timeout int
        timeout (default 60)
  -version
        print version information
```
